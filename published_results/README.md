# Published numerical data

This directory contains the numerical data used for the tables and figure in
Section 6 of the associated article.

- `section_6_2_convergence/` contains the `.npz` level data and the resulting
  `*_eoc.csv` tables for all eight convergence configurations.
- `section_6_3_projection_stability/` contains the stability CSV and the
  generated strong/multiplier figure panels.

New computations should be written to `results/`, not to this directory. The
`results/` directory is ignored by Git.
